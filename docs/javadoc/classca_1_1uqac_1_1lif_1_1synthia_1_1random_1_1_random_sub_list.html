Create a new RandomSubList picker based on a given list.

If this list is empty, the method will return a NothingPicker. The new instance will also have the same internal states for the m_listReducer, m_pick and m_asIs attributes as the original one.

Parameters
elementThe list used by the new RandomSubList picker to produce sublists.
Returns
The new instance of the class or a NothingPicker.
Warning
Depending of the content of the list, it is possible that the returned object is a NothingPicker. If a RandomSubList is returned, it is also possible that some of the sublists generated by the pick() method of this instance could generate ca.uqac.lif.synthia.NoMoreElementException's.
