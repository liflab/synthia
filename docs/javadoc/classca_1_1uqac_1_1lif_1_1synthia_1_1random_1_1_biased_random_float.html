ca.uqac.lif.synthia.random.BiasedRandomFloat Class Reference

Detailed Description

Generates pseudo-random floating-point numbers in the interval [0,1], but according to a probability distribution that is not uniform.

Concretely, a value is first picked uniformly, and is then multiplied by a factor β called the bias. The resulting value is then replaced by 0 or 1 if it lies outside the interval. Setting β < 1 results in a picker that favors values closer to 0, while setting β > 1 results in a picker that favors values closer to 1.
