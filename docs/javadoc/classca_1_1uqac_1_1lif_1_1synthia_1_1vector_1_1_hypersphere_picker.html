Detailed Description

Generates n-dimensional vectors with a given modulus.

This picker does so by taking a positive value r and n-1 pickers providing angles θ1, … θn-1 (in radians). These values are taken as the n-dimensional polar coordinates of a point, which is then turned into an n-dimensional vector in Euclidean coordinates, as is defined in this paper:

A. Muleshkov, T. Nguyen. (2013). Easy proof of the Jacobian for the n-dimensional polar coordinates. (Paper)

The end result is that this picker generates vectors corresponding to points that lie on the surface of an n-dimensional hypersphere of some radius r.
