Detailed Description

Picker who implements Bounded.

This picker enumerates all the possibility of of combinaisons of picked value from an array of Enumerative pickers. For example, an AllPickers containing an array of 2 AllBooleans will generates one array in the following order :

  1. [<Boolean>false</Boolean>, <Boolean>false</Boolean>]
  2. [<Boolean>true</Boolean>, <Boolean>false</Boolean>]
  3. [<Boolean>false</Boolean>, <Boolean>true</Boolean>]
  4. [<Boolean>true</Boolean>, <Boolean>true</Boolean>]

After that, the picker will throw a NoMoreElementException if the pick method is called one more time.
