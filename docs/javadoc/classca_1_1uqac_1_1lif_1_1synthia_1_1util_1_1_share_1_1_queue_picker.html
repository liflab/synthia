ca.uqac.lif.synthia.util.Share< T >.QueuePicker Class Reference

Detailed Description

A picker that acts as a proxy for the values obtained by the parent Share picker.

Each instance of QueuePicker is guaranteed to output the same values in the same order as the picker given to Share when instantiated.
