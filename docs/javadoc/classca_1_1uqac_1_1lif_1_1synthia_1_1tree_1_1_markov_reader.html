ca.uqac.lif.synthia.tree.MarkovReader< T, U > Class Template Reference

Detailed Description

Converts a graph into an equivalent Markov chain.

Each vertex of the graph is turned into a state of the Markov chain by overriding method getPicker(). Transitions in the chain mirror the adjacency relation of the graph, assigning equal probability to each outgoing edge.
