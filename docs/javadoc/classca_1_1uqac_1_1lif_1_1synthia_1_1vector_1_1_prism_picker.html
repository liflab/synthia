ca.uqac.lif.synthia.vector.PrismPicker Class Reference
API Documentation

Detailed Description

Generates a vector by independently picking a value for each of its coordinates.

If each of the pickers define a possible range of values, this picker amounts to generating points inside a multi-dimensional "prism" –hence its name.
