API Documentation

Detailed Description

Picks an object based on the value of a random number generator.

All descendants of this class use an instance of Java's Random class. They also implement the Seedable interface, which means that this internal generator can be seeded with some value; furthermore, a call to reset() will instantiate a new instance of Random with the initial seed. This means that resetting a RandomPicker should in principle make it output the same sequence of values as when it was first created.
