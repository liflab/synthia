API Documentation

Detailed Description

Applies an affine transform to a value produced by another picker.

It takes two parameters m and b. If x is the value produced by the picker, the affine transforms returns mx+b.

This class is abstract; it has two concrete descendants, AffineTransformInteger and AffineTransformFloat.

Parameters
<T>The type of number produced (i.e. Float, Integer, etc.)
