◆ checkIfInfiniteLoop()

boolean ca.uqac.lif.synthia.relative.PickIf< T >.checkIfInfiniteLoop ( int  iteration_counter)
protected

Protected method to check if the instance of the class detects that the while loop of the pick() method fell into an infinite loop by supposing that if the loop exceed a maximal number of iteration, it will cause an infinite loop.

Warning
If m_maxIteration is negative, the pick method will not check if the iteration counter has exceeded the value of m_maxIteration and this method will always return false. This means that it's possible that calling pick causes an infinite loop. Only use a negative value for m_maxIteration if you assume that calling pick could cause an infinite loop.
Returns
true if infinite loop is detected and false if it's not the case.
