Detailed Description

Generates a sequence of objects by a random walk in a Markov chain.

A Markov chain is a stochastic model describing a sequence of possible events. Each state of the chain is associated to a Picker; transitions between states A → B are labelled with the probability of going to B when the current state ia A. The following picture shows an example of a Markov chain:

<T>The type of objects returned by the Markov chain. The pickers associated to each state must return objects of type T or its descendants.
