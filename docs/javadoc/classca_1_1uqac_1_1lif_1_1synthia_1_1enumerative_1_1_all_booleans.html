API Documentation

Detailed Description

Picker who implements EnumerativePicker.

This picker enumerates the boolean value (<Boolean>false</Boolean>, <Boolean>true</Boolean>) and throws a NoMoreElementException if the picker picks another value after the picker has finished to enumerate the Boolean values.
